Личный кабинет инвестора

Удобный и безопасный сервис для взаимодействия акционера и регистратора.

Сервис предоставляет инвестору следующие возможности:

  • просмотр сведений о ценных бумагах, учитываемых на лицевых счетах акционера, открытых в реестрах, ведение которых осуществляет Регистратор, в том числе об их количестве, номинальной и/или рыночной стоимости, типе, государственном регистрационном номере;
  • просмотр сведений об операциях по лицевым счетам;;
  • просмотр сведений об обременениях, с подробной информацией по каждому;
  • просмотр информации о начисленных и выплаченных дивидендах по эмитентам, в отношении которых Регистратор выполняет функции агента по выплате доходов по ценным бумагам;
  • просмотр информации о завершенных и текущих общих собраниях акционеров, где у акционера имеются ценные бумаги;
  • дистанционное голосование на общем собрании акционеров эмитентов, которые в соответствующем соглашении с Регистратором, предусмотрели право дистанционного голосования по принадлежащим акционеру ценным бумагам;
  • направление Регистратору запросов на получение информации;
  • получение ответов на отправленные Регистратору запросы (выписки со счета зарегистрированного лица, справки о наличии ценных бумаг на счете, отчета об операциях по счету за период);
